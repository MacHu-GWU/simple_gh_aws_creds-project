Install
------------------------------------------------------------------------------

``simple_gh_aws_creds`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install simple-gh-aws-creds

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade simple-gh-aws-creds
